# Event Manager C コピペ手順

## 1. フォルダ作成

任意の場所に `event-manager-c` フォルダを作成し、この一式をそのまま配置します。

## 2. ターミナルを開く

```powershell
cd <展開したフォルダ>\event-manager-c
```

## 3. 依存関係を入れる

```powershell
npm install
```

## 4. 開発サーバ起動

```powershell
npm run dev
```

通常のURL:

```text
http://localhost:5173
```

## 5. 本番ビルド確認

```powershell
npm run build
npm run preview
```

通常のURL:

```text
http://localhost:4173
```

## 6. E2Eテスト

最初の1回だけブラウザを入れます。

```powershell
npx playwright install chromium
npm run test:e2e
```

## 7. GitHubへ公開

```powershell
git init
git add .
git commit -m "Initial commit for Event Manager C"
git branch -M main
git remote add origin https://github.com/<GitHubユーザー名>/event-manager-c.git
git push -u origin main
```

その後 GitHub の `Settings > Pages` で `main / root` を選択します。

公開 URL 例:

```text
https://<GitHubユーザー名>.github.io/event-manager-c/
```
